Instructions

1. Open Adobe Photoshop CC 2015.5

2. Create a new file and name it Design.

Document Type: International Paper

Size: A4

Resolution: 720

Color Mode: RGB Color

3. Select this black and white texture (http://static8.depositphotos.com/1387950/864/i/950/depositphotos_8648992-Black-and-white-scratches-texture.jpg ) from the internet and paste it (covering the entire canvas).

4. Draw a black rectangle (fill: black & border: black) in the center, leaving a one-inch border.

5. Select this black and blue texture (http://img00.deviantart.net/69ab/i/2009/298/7/3/blue_black_scratched_texture_by_beckas.jpg )from the internet and paste it on top of the black rectangle. The length should be from 1.2-10.3inches and the width from1.1-7inches.

6. Select the horizontal type tool, size 100pt, color white and write the word “DESIGN” between 9-10inches of the canvas.

7. Save this file as a psd.
